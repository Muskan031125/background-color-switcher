
# Background Color Switcher

A simple interactive web project that changes the webpage background color when a user clicks on a color square.

## Features
- Clickable color squares
- Instant background color change
- Clean and responsive layout
- Event-driven interaction using JavaScript

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
Each color block has a unique ID representing a color.  
When a user clicks a block, JavaScript reads the clicked element's ID and applies it as the page's background color.
